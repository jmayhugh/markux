## Task 2: Update `src/url.js` to drop all query params

**Files:**
- Modify: `src/url.js`

- [ ] **Step 1: Replace the file contents**

Replace the entire contents of `src/url.js` with:

```js
export function normalizeUrl(rawUrl) {
  const url = new URL(rawUrl);
  url.hostname = url.hostname.toLowerCase();
  url.hash = "";
  url.search = "";
  let result = url.toString();
  if (result.endsWith("/") && new URL(result).pathname !== "/") {
    result = result.slice(0, -1);
  }
  return result;
}
```

Key changes vs. old file:
- `TRACKING_PARAMS` constant deleted.
- The `URLSearchParams` loop that deleted specific keys is replaced with one line: `url.search = "";`.
- Everything else (hostname lowercase, hash strip, trailing-slash trim, root-path exception) is unchanged.
